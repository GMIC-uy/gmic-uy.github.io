GMIC: Grupo de Modelación Integrada de Cuencas

GMIC es un espacio de colaboración interinstitucional que desarrolla y aplica modelos para analizar la cantidad y calidad del agua, apoyar la planificación y evaluar escenarios de gestión en cuencas hidrográficas del Uruguay.

About GMIC

This page organizes GMIC information into a format that is easy to maintain on GitHub Pages without a backend.

What this page does

  • Simple project description
  • Searchable model metadata table
  • Interactive map of institutions and case studies
  • Cards for each model with links and keywords

How to update it

Edit one JSON file with model metadata. The table, cards, statistics, and map update automatically.

You can also add GeoJSON files for catchments and connect them through the map_link field.
